# HR-аналитика: смена работы специалистов по обработке данных

Инструменты: Python, Pandas, Google Colab.

Работа с данными и анализ содержатся в файле HR_Analytics.ipynb.

## Исходные данные

Набор данных [HR Analytics: Job Change of Data Scientists](https://www.kaggle.com/datasets/arashnic/hr-analytics-job-change-of-data-scientists) размещен на kaggle.

Этот набор содержит информацию о специалистах, которые прошли курсы компании и желают или не желают сменить место работы.
Присутствует информация об индексе развития города, об образовании, об опыте, о текущей компании кандидата. В наборе содержится 1 таблица.

### Поля таблицы

- enrollee_id: уникальный идентификатор кандидата;
- city: код города;
- city_development_index: индекс развития города (масштабированный);
- gender: пол кандидата;
- relevent_experience: соответствующий опыт кандидата;
- enrolled_university: тип зачисленного курса университета, если таковой имеется;
- education_level: уровень образования кандидата;
- major_discipline : основная дисциплина образования кандидата;
- experience: общий опыт кандидата в годах;
- company_size: количество сотрудников в компании текущего работодателя;
- company_type : тип текущего работодателя;
- last_new_job: разница в годах между предыдущей и текущей работой;
- training_hours: часы обучения завершены;
- target: 0 – не ищу смены работы, 1 – ищу смену работы.

## Импорт и очистка данных

Данные были импортированы в виде архива в Google Colab, после чего разархивированы. В наборе содержатся 19158 строк и 14 столбцов.
В данных не обнаружены дубликаты и выбросы, но присутствует достаточно большое количество пропусков в нескольких столбцах.
Максимальное количество пропусков в записях о типе и размере компании, а также о поле специалистов.

## Распределение данных

После изучения содержания столбцов можно сделать некоторые умозаключения.

1. В наборе данных количество мужчин (13221) существенно выше количества женщин (1238), а также присутствует много пропусков (4508).
2. Наибольшее количество записей приходится на выпускников университетов. Их количество в разы выше любой другой группы.
3. В датафрейме представлена достаточная информация по компаниям разного размера, однако в столбце много пропусков.
4. Столбец с опытом работы разделен на группы для удобства анализа. Людей с малым опытом (5020) гораздо больше, чем в остальных группах.
5. Столбец с индексом развития города также разделен на группы. В наборе больше всего записей по городам с низким индексом развития города, а с высоким в 4 раза меньше.
6. Данные о разности между текущем месте работы и предыдущем переведен в числовой формат. Основные записи относятся к интервалу в один год, которые занимают почти половину массива данных.
7. Большинство специалистов не хотят менять место работы после прохождения курсов. Отношение желающих сменить место к не желающим - один к трем.

## Анализ данных

1. Обнаружена прямая зависимость между некоторыми переменными: индексом развития города и опытом специалиста, индексом развития города и годами между текущей и следующей работой, опытом кандидата и годами между работами.
   Чем выше показатели одной, тем выше показатели другой.
2. С помощью показателя phik обнаружена зависимость между переменной target и city_development_index. Чем выше группа индекса развития города, тем ниже среднее значение желания специалистов сменить работу.
3. Число часов обучения и среднее значение показателя желания смены работы не заисят от пола специалиста. Эти категории имеют примерно одинаковые значения для разных групп.
4. Больше всего работу хотят сменить выпускники университетов. Для остальных групп желание смены работы в среднем ниже.
5. Желание сменить работу также зависит от количества лет после прошлой работы. Чем дольше специалист находится на текущем месте работы, тем меньше желания её сменить.

## Выводы

В данном проекте проведен анализ данных по специалистам, прошедшим курсы компании. Изучены доступные характеристики кандидатов, а также проанализировано их влияние на желание сменить текущее место работы.
Сделанные в ходе изучения умозаключения могут использоваться для определения специалистов, которые планирую перейти на работу в данную компанию.
